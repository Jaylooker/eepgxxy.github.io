Web Design Workflow.

  • Wireframing
  • UI design
  • Prototyping

Basic Programming languages - HTML

  • HTML is supposed to do the backbone of the web page.
  • HTML should be used according to their semantic meanings.
  • Styling the page should not be the purpose of HTML.

Basic Programming languages - CSS

  • CSS is better used as a separate file from the HTML file.
  • CSS selectors should be appeared in the file the same sequence as they shown in the HTML file.
  • CSS selectors should be written with the future-proof in the mind

Basic Programming languages - JavaScript

  • JavaScript is better used as a separate file from the HTML file.
  • JavaScript functions and objects should be used if possible.

Basic Programming languages - Java

  • Servlet is HTML in java.
  • JSP is java in HTML.
  • both Servlet and JSP are compiled to java classes before they are sent to the client.
